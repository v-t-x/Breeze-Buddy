# Breeze Buddy

Temperature-controlled wearable cooling device built with Arduino.

## Components
- Arduino microcontroller
- TMP36 temperature sensor
- DC fan
- battery power system

## Features
- Automatically activates cooling when temperature exceeds threshold
- Portable wearable design
- Low power consumption

## Technologies
- Arduino (C++)
- Circuit design
- Sensor integration
